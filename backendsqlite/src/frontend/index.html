Déploiement de votre Frontend

Une fois votre frontend déployé, ce fichier sera masqué par votre application présente dans dist/ et servi directement depuis ce même endroit.

Note: ce fichier fait partie de la configuration initiale du projet. Il est utilisé comme page d'accueil par défaut pendant le développement, mais une fois le build effectué, il sera remplacé par votre application React servie par Express.

Pour construire votre application, exécutez simplement la commande npm run build dans votre terminal.
